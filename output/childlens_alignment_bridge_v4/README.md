# ChildLens bridge v4

This directory is the privacy-safe public record for the separately versioned
transcription-free audiovisual development preflight.

- `protocol_and_split_freeze_receipt.json` binds the protocol, code, public
  frontend weights, immutable v1–v3 trees, development-only manifests, frozen
  12/6 participant split, windows, and controls before media decoding.
- `calibration_summary.json` reports K-safe participant-clustered
  model-independent distributions for all 18 development participants.
- `development_go_no_go_report.{json,md}` separates training from
  participant-disjoint evaluation and reports both required control envelopes.
- `validation_receipt.json` is the v4.0.0 execution validation.
- `public_privacy_correction_v4_0_1.json` and
  `validation_receipt_v4_0_1.json` record the public-only correction that
  collapses suppressed category labels. Restricted results and the terminal
  decision were unchanged.

The result is a terminal development `NO_GO`: preprocessing was stable, but
real pairing did not show the preregistered participant-clustered meaningful
advantage over the primary within-recording time-shift control. The 22 locked
participants remained sealed, and no simulator or physical side-cue conditions
were run.

All identifiers, exact intervals, media, embeddings, projection weights, and
row-level scores remain in the owner-private local ChildLens quarantine.
